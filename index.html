Projects

NineTracks

A single-page, fully responsive React/Redux music playlist app built on RESTful Rails architecture. When a user selects a playlist, it holds the selected playlist in application state, independent of an asynchronous call, allowing the user to use the site while playing a song without interruption. The image and song storage are hosted on Cloudinary.

Cine-Reel

Single page visual representation of Michael Bach's Color Wheel Illusion. A wheel of dots is placed, each dot level has 12 dots separated by 30 degrees (360 degrees total). The Cine-Reel utilizes vanilla JavaScript for the controls and D3.js for SVG object placement and attribute alteration.

QT-Kompressor

QT-Kompressor is a single-page application utilizing React/Redux, Node.js, and MongoDB to demonstrate the workings of the quadtree data structure. By recursively decomposing an image into quadrants by similar colors, it can render a compressed version of the image. When breaking down based on looser thresholds, these flattened regions are illustrative of the underlying structure.
